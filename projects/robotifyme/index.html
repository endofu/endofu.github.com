robotify.me

  • robotify.me - Live site
  • BBH Labs posts about the project

At BBH Labs we wanted to work on a project that would reflect people’s social behaviour back to them in an engaging way. We also wanted to try a different team structure and process on a bigger Labs experiment. We came up with robotify.me.

robotify.me analyses people’s social personae and send them back something more interesting than the Klout score - a little robot who changes weekly.

It’s a Django project on Google App Engine. The front-end is a responsive single-page web application built with backbone.js, require.js, handlebars and Twitter’s Bootstrap. It communicates with the backend through a little API that is planned to be opened in the future. At the moment it uses data from Facebook, Twitter and Foursquare but it’s been desinged in a way that allows adding more platforms easily. I used PIL to compose the robots on the backend and to find out the dominant colours of the user’s profile images. The data collection, parsing, textual and quantitative analysis was a very good opportunity to practice Python’s functional features, like lambdas, list comprehensions and generators.
