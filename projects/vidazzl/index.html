Vidazzl

  • Vidazzl - Live site

This idea came out of a very simple challenge: when you’re John Hegarty and you don’t “do” Powerpoint or Keynote, how can you create theatre, context and visual stimulus around a big talk? This BBH Labs experiment aims to use voice recognition to provide ‘fuzzy’ contextual visual background to speeches.

I built the prototype of the user interface while we were looking for the provider of a solid voice recognition technology. In this version, the video has been manually tagged with keywords, and while the user listens to the video, the Javascript frontend displays in real time the latest tweets, Wikipedia articles and flickr images to the corresponding time-tagged keywords and animates them using the canvas renderer of three.js. I wrote another version that used WebGL and a GLSL pixelshader for the depth of field to make it look prettier.
